# Security Policy

## Supported Versions

Only the latest major version is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < 1.x   | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you find a security vulnerability, please do NOT create a public issue. Instead, please report it via the following steps:

1. Send an email to the maintainer (see `package.json`).
2. Provide as much detail as possible, including steps to reproduce.
3. Give us a reasonable amount of time to respond and fix the issue before making it public.

Thank you for helping keep this project safe!
